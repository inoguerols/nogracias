Cómo funciona

Dos capas, mínimo coste, y nunca borra.

Cada correo recorre un embudo. Lo barato y seguro primero; la IA solo para lo dudoso.

1

Lista de confianza

¿De un dominio tuyo, un proveedor o tu banco? Se queda en la bandeja. Gratis e instantáneo.

2

Reglas (gratis)

Puntúan cada correo por señales: IA, captación, competencia, reseñas, fichajes, SEO/Google, marketing/Meta, ganchos de pregunta, "Re:" falso… Ninguna señal marca sola: hacen falta ≥2 combinadas, así no se come correo legítimo.

3

Capa IA opcional

Solo para lo que las reglas no tienen claro: un modelo lee el correo y juzga por sentido "¿venta no solicitada? sí/no". Caza fraseos nuevos y en cualquier idioma, sin perseguir palabras.

4

A una carpeta — nunca borra

Lo marcado se mueve a una carpeta aparte que revisas cuando quieras. Si algo se equivoca, lo recuperas. Cero correos importantes perdidos.

Validado contra un buzón real: 0 falsos positivos sobre 30 correos legítimos (pacientes, proveedores, gestoría, banco).

Dónde se instala

Funciona en cualquier correo.

Una sola lista de reglas (rules.json) genera los instaladores de cada servicio. Lo importante es dónde se entrega tu correo, no qué app usas para leerlo.

Tu correo está en…Cómo se instalaLleva IA
Servidor con Sieve (Plesk, cPanel, Fastmail, mailbox.org, Migadu…)Pegar nogracias.sieve en los filtros del webmail—
Gmail / Google WorkspaceImportar gmail-filters.xml—
Outlook / Microsoft 365 o cualquier IMAPfilter.py (cron / GitHub Actions / Docker)✅

Los filtros del servidor (Sieve/Gmail) son siempre-activos pero solo por reglas. La capa IA vive en filter.py.

Capa IA opcional

Trae tu modelo. O ninguno.

Apagada por defecto. Habla el formato estándar de OpenAI con la librería de serie de Python (sin SDKs), así que vale cualquier proveedor — o un modelo local.

Privacidad y seguridad

Pensado para no fastidiarla.

  • Mueve o etiqueta, nunca borra. Nada se pierde.
  • Sin dependencias. Solo Python de serie; auditas el código en un rato.
  • Tus credenciales se quedan en local (un .env que no se sube) o en secrets cifrados.
  • Con IA local, el correo no sale de tu servidor.
  • Lista de confianza para que nunca se toque correo de pacientes, proveedores o tu banco.

Empieza en un minuto:

git clone https://github.com/inoguerols/nogracias
cd nogracias
python3 filter.py --self-check     # prueba el clasificador, sin tocar tu correo
